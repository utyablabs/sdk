## Installation

Install the UtyabSwap SDK with your preferred package manager.

### Using npm
```bash
npm install @utyablabs/sdk
```

### Using Yarn
```bash
yarn add @utyablabs/sdk
```

### Using pnpm
```bash
pnpm install @utyablabs/sdk
```

### Using Bun
```bash
bun add @utyablabs/sdk
```
